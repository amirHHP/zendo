Zendo is an ultra-minimalist, distraction-free GTD task manager Chrome extension. It structures your inbox, categorizes items with collapsible project trees, and refines tasks using Gemini AI. زندو یک اکستنشن مدیریت تسک فوق‌العاده مینیمال و بدون حواس‌پرتی بر پایه متدولوژی GTD است. زندو تسک‌های ورودی را دسته‌بندی کرده و با استفاده از هوش مصنوعی Gemini آن‌ها را بسط می‌دهد.

Get started in minutes راه‌اندازی در چند دقیقه

Setting up Zendo is straightforward. Follow these steps to unlock deep, focused organization. شروع کار با زندو بسیار ساده است. مراحل زیر را برای دسترسی به محیط تمرکز عمیق طی کنید.

01

Download and Load دانلود و بارگذاری اکستنشن

Clone the repository or download the ZIP, open chrome://extensions, enable Developer Mode, and click Load Unpacked. پروژه را کلون یا فایل ZIP آن را دانلود کنید. آدرس chrome://extensions را باز کرده، حالت توسعه‌دهنده را فعال کنید و دکمه Load Unpacked را بزنید.

02

Add Gemini API Key وارد کردن کلید API

Get a free key from Google AI Studio. Open settings in Zendo, paste your key, and select a Gemini model (e.g. Gemini 1.5 Flash). یک کلید API رایگان از Google AI Studio بگیرید. بخش تنظیمات زندو را باز کرده، کلید خود را جایگذاری کرده و یکی از مدل‌های Gemini را انتخاب کنید.

03

Capture & Organize ثبت و تمرکز

Add items to your inbox, create nested projects, and use AI features to elaborate complex tasks into actionable check-lists. کارهای روزانه را در صندوق ورودی ثبت کرده، به پروژه‌های درختی انتقال دهید و با کمک هوش مصنوعی کارهای پیچیده را به مراحل ساده بشکنید.
